In this code: We define a textarea element with the ID highlightedTextarea and some sample text. There's a "Highlight" button that, when clicked, triggers the highlightText JavaScript function. Inside the highlightText function, we get the selected text within the textarea and wrap it with a element with the highlighted class, which defines the background color for highlighting. We then replace the selected text with the highlighted version by updating the textarea content. When you select text within the textarea and click the "Highlight" button, the selected text will be highlighted with a yellow background. You can modify the CSS styles and highlight color to suit your design preferences.
